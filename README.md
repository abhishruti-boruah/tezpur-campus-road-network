# tezpur-campus-road-network

**# Tezpur Campus Road Network Graph

This project models the road network of Tezpur University using graph representations.

Buildings are treated as nodes and roads are treated as edges.

## Features

- Graph representation of campus
- Adjacency matrix implementation
- Incidence matrix implementation
- Manual edge creation between buildings

## Files

main.py – runs the program  
graph.py – graph implementation  
data.csv – list of campus buildings  

## How to Run

```bash
python main.py**
